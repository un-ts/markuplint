# markuplint-angular-parser

Angular parser for markuplint.

## Usage

```sh
# yarn
yarn add markuplint-angular-parser

# npm
npm i markuplint-angular-parser
```

Add `parser` option into your `.markuplintrc.*` file.

```json
{
  "parser": {
    ".html$": "markuplint-angular-parser"
  }
}
```
